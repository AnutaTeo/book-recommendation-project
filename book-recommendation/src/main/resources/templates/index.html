Semantic Web Book Recommendation System

This web application uses RDF/XML, Apache Jena, OWL, SPARQL, and RAG-based chatbot features.

Features

  • Upload RDF/XML file
  • Visualize RDF graph
  • Add and modify books using RDF
  • List books and view book details
  • Use OWL ontology and SPARQL queries
  • Use chatbot enhanced with book data
